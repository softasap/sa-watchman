sa-watchman
===========

[![Build Status](https://travis-ci.org/softasap/sa-watchman.svg?branch=master)](https://travis-ci.org/softasap/sa-watchman)

Facebook's watchman

Example of usage (all parameters are optional)

Simple

  roles:
    - {
        role: "sa-watchman"
      }


Advanced:


  roles:
    - {
        role: "sa-watchman",
        watchman_version: 4.5.0,
      }
